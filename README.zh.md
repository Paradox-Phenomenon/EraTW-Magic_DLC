### eratw 画蛇添足开发者指南

> *本来是没人写这东西的，还是因为一些事情单独拉了一个私有开发仓库，主仓库多余分支和权限都清掉了，毕竟是公开项目还是要说明一下。*

本着最小化权限的原则（主要是好多为爱发电的小伙伴不会用 git 经常搞出来事故），删除了除主分支外的所有分支及大部分手动添加的成员，并已转移至开发仓库。如需提交代码可以直接向 [主仓库](https://gitgud.io/era-games-zh/touhou/eratw-sub-modding) 的主分支提交 [merge request](https://gitgud.io/era-games-zh/touhou/eratw-sub-modding/-/merge_requests) 或申请加入开发组。

考虑到大部分贡献者是为爱发电的非专业技术人员，不对 merge request 做出过多限制，为协作开发指定的开发组内部规范也不在此公开。~~所以搞技术的就不要找了~~
